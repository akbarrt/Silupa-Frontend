**SiLupa (Sistem Lumbung Panen)** is a web-based application designed to digitize and streamline the management of agricultural harvest storage, inventory, distribution, and reporting.

The system aims to improve transparency, efficiency, and data accuracy in post-harvest management by providing a modern, scalable, and user-friendly digital platform.

---

## ğŸŒŸ Overview

Agricultural harvest management often relies on manual records, which can lead to data inconsistency, inefficiency, and lack of transparency.  
SiLupa addresses this problem by offering a centralized digital dashboard that enables stakeholders to monitor harvest stock, storage capacity, and distribution activities in real time.

This repository currently contains the **frontend application**, built with modern web technologies and designed to be easily integrated with a backend API.

## âš™ï¸ Getting Started

### Prerequisites

> The following steps require [NodeJS](https://nodejs.org/en/) to be installed on your system, so please
> install it beforehand if you haven't already.
- Node.js 18+ recommended
- `npm` or `yarn` package manager

### Install dependencies

```bash
cd silupa
npm install
```

### Run the development server

To get started with your project, you'll first need to install the dependencies with:

```bash
npm install
```

Then, you'll be able to run a development version of the project with:

```bash
npm run dev
```

After a few seconds, your project should be accessible at the address
[http://localhost:5173/](http://localhost:5173/)

---

### Build for production

```bash
npm run build
```

The optimized output will be generated by Vite in the default `dist` folder.

## Development Notes

- Routing is configured in `src/App.tsx` using `react-router-dom` with an `AppShell` layout route.
- UI elements in `src/components/ui` wrap Radix primitives; prefer using these instead of raw Radix components to keep styling consistent.
- Charts are implemented with `recharts` and are primarily present in dashboard and reports components.
